USAGE:
======

* First include the start file:

?>

* Use the library (check the documentation):

<?php

include_once("msc_lib/start.php");

m_config_var("get_vars",$_GET ? m_stripslashes($_GET) : false);

print_r(m_config_var("get_vars"));

?>
