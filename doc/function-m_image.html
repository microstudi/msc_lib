Function m_image

Creates and resize a image if needed, returns image data or the gd resource

Example

// This will flush the image resized at 200x150 by cropping width or height if necessary
m_image('my_file.jpg', 200, 150, 1);
die;

Parameters summary
mixed $file original file to process, could be a string location, url, gd resource or a mImage class
mixed $width = 0 new width of the file to be returned (leaving empty or 0 means the original width)
mixed $height = 0 new height of the file to be returned (leaving empty or 0 means the original height)
mixed $proportional = 1 0: the image will be resized to the specified w/h without keeping aspect ratio
1: the image will be resized to the specified w/h keeping aspect ratio (by cropping width or height)
2: the image will be resized to the max w/h keeping aspect ratio (without cropping)
3: the image will be resized to the min w/h keeping aspect ratio (without cropping)
mixed $return = 'flush' flush returns the new image directly (as GD methods imagejpeg(), imagepng() do)
gd returns the gd resource (doesn't apply the cache)
mimage returns the mImage resource (doesn't apply the cache)
quality $quality = 90 JPEG quality (PNG or GIF does not uses this parameter)
Return value summary
mixed Returns according the specified $return value
