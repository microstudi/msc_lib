Class mImage

Image Manipulation class

This class provides methods to resize, convert, save & flush images. This class is used by the file functions/images.php

Example

$img = new mImage("file.jpg");
$img->set_proportional(2);
$img->set_quality(90);
$img->resize(200,100);
$img->flush();
