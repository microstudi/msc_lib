Detailed Description

<select> tag for the default HTML5 view

Author
Ivan Verg√©s

Example:

echo m_view("select",array('id'=>"select1",'class'=>"my_class",'body'=>"SELECT html content"));
//or
$items = array();
$items[] = "Option 1 text";
$items[] = "Option 2 text";
echo m_view("select",array('id'=>"select1",'class'=>"my_class",'items'=>$items));
//or
$items = array();
$items[] = array('id'=>"option1",'value'=>"option_1_value",'text'=>"Option 1 text");
$items[] = array('id'=>"option2",'value'=>"option_2_value",'text'=>"Option 2 text");
echo m_view("select",array('id'=>"select1",'class'=>"my_class",'items'=>$items));

Parameters
divinputencapsulates <input> in a div with the class specified
namehtml name
disabledhtml label disabled (true or false)
labelAdds the <label>...</label> before the <select>, if param id is present adds <label for="...id...">
endlabelIf true, the <label> goes after the <select>
labelidid for the <label id="...">
labelclassclass for the <label class="...">
bodyhtml content inside <select>...</select>
optionsa alias for items
itemsif body not exists and this is a array, it will be used to display the internal <option> elements of <select>, a the same time items could be a array with text or a sub-array with id, class, style, value & text sub-parameters
