Detailed Description

<input type="radio"> tag for the default HTML5 view

Author
Ivan Verg√©s

Example:

$items = array();
$items[] = array('label'=>'Label 1','id'=>"item1",'value'=>"value1",'class'=>"my_class");
$items[] = array('label'=>'Label 2','id'=>"item2",'value'=>"value2",'class'=>"my_class");
echo m_view("radios",array('options'=>$items,'addbr'=>true));

Parameters
addbradds a <br> tag between the <input> tags
optionsa array of items (every <input>) with the different options:
Items parameters:
  • id id of the <input id="">
  • name name of the <input name="">
  • class class of the <input class="">
  • value value of the <input value="">
  • disabled if(true) adds <input disabled="disabled">
  • label Adds the <label>...</label> before the <input>, if param id is present adds <label for="...id...">
  • endlabel If true, the <label> goes after the <input>
  • labelid id for the <label id="...">
  • labelclass class for the <label class="...">
