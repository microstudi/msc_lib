Class mFile

File wrapper manipulation class

This file is used to upload, download on several services like, local, ftp, ssh2, AmazonS3. This class is used by the file functions/file.php

Example:

$db = new mFile('type','host','port','username','password','path');
$db->connect();
$db->upload($local_file, $remote_file);
